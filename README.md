🪨📄✂️ Jokenpo
----

Projeto desenvolvido em linguagem C com o objetivo de praticar:
Estruturas de decisão (if, else, switch)
Estruturas de repetição (do while)
Geração de números aleatórios (rand, srand)
Lógica de programação
----

🎮 Sobre o Projeto
Este é um jogo de Jokenpo (Pedra, Papel e Tesoura) com:

✅ Menu interativo
✅ Exibição das regras
✅ Computador com jogada aleatória

O jogador disputa contra o computador.

---
🏆 Regras do Jogo

Pedra ganha de Tesoura
Tesoura ganha de Papel
Papel ganha de Pedra
Escolhas iguais resultam em empate
